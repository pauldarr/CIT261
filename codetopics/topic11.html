What is this example?

This is a basic blog type webpage. It's a combination example that uses Transitions, Transformations, and Animations. The hamburger menu uses Transitions and transforms. The slideshow uses Transition and Animation all in CSS

What does it do?

Click on the Menu Icon to transform it to "X" and bring down a menu. The home option is set for my home here. Hovering over the slideshow pauses it, displays a tooltip, and pauses the progress bar.
